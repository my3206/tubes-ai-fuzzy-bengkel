**Cara Menjalankan Program:**

1. Pastikan Python sudah terinstall di komputer.
2. Pastikan file berikut berada dalam satu folder:
   - fuzzy_bengkel.py
   - bengkel.csv
   - README.txt
3. Buka terminal atau command prompt pada folder tersebut.
4. Jalankan program dengan perintah:
   python fuzzy_bengkel.py
5. Program akan memproses data bengkel menggunakan Sistem Fuzzy Mamdani.
6. Hasil berupa skor kelayakan setiap bengkel akan disimpan dalam file:
   hasil_fuzzy.csv
7. Program juga menampilkan 5 bengkel terbaik pada layar.
